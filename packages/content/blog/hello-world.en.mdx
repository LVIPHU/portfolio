Welcome to my blog!

This is a sample post so you can see how the blog works. Each post is an MDX file in `packages/content/blog`, named `<slug>.<locale>.mdx` — this file is `hello-world.en.mdx`, and the Vietnamese version is `hello-world.vi.mdx`.

## What can you write in MDX?

Everything Markdown supports:

- **Bold**, _italic_, `inline code`
- Lists, quotes, links
- Syntax-highlighted code blocks:

```ts
const greeting = 'Hello!'
console.log(greeting)
```

> MDX also lets you embed React components if you ever want to level up.
