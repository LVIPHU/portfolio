Chào mừng bạn đến với blog của mình!

Đây là bài viết mẫu để bạn thấy blog hoạt động thế nào. Mỗi bài là một file MDX trong `packages/content/blog`, đặt tên theo dạng `<slug>.<locale>.mdx` — ví dụ file này là `hello-world.vi.mdx`, còn bản tiếng Anh là `hello-world.en.mdx`.

## Viết được gì trong MDX?

Mọi thứ Markdown hỗ trợ:

- **In đậm**, *in nghiêng*, `inline code`
- Danh sách, trích dẫn, link
- Code block có syntax:

```ts
const greeting = "Xin chào!";
console.log(greeting);
```

> MDX còn cho phép nhúng component React nếu sau này bạn muốn nâng cấp.

Xóa file này đi và bắt đầu viết bài của riêng bạn nhé!
